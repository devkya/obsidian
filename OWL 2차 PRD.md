## 데스크탑 앱
### 대본 등록

| 기능            | 설명                                                                                          | 비고                                 |
| ------------- | ------------------------------------------------------------------------------------------- | ---------------------------------- |
| 대본 업로드        | 1. LLM을 활용한 PDF 자동 파싱<br>2. 양식에 맞는 엑셀로 자동 파싱                                                | LLM 과금 대상                          |
| 엑셀 양식 다운로드    | 사용자가 엑셀 양식을 다운로드 할 수 있어야함                                                                   | 메모 설명                              |
| 대본 최적화        | 메인 언어 기준 최적화<br>- 메인 언어와 번역 언어가 한 페이지에서 의미만 동일하면 됨<br>- 한국어 1줄 / 영어 2줄 가능<br>- 다국어는 3줄까지 허용 |                                    |
| 대본 업로드 상태     | 배치 작업으로 작업 상태에 대해 사용자가 알 수 있도록 대본 리스트 화면에서 상태를 표시해야 함<br>- 업로드중 / 완료 등                      |                                    |
| 완료 후 수정 필요 안내 | 모든 작업이 완료되면, 사용자 수정을 유도해야 함<br>사용자 직접 검토 강조                                                 |                                    |
| 대본 추출         | 텍스트 수정 & 번역 언어 추가를 위한 엑셀 추출<br>인덱스(page, line, block) 포함 추출<br>행 삭제 & 추가되어서는 안됨             | 메모 설명 추가                           |
| 다국어 추가        | 1. LLM을 활용하여 번역 PDF 자동 파싱<br>2. 추출된 대본(엑셀) 업로드 -> 유효성 검사                                    | LLM 과금 대상<br>번역 PDF 자동 파싱 성능 검증 필요 |
| AI 번역 기능      | AI 번역 기능<br>사용자에게 번역 완료 후 사용자 직접 검토 강조<br>                                                  | 번역 오류 가능성 최소화를 위해 길이 제한을 두지 않음     |
### 대본 수정

| 기능                   | 설명                                                                                                 | 비고                                                                     |
| -------------------- | -------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| 미리보기 화면              | 엑스리얼 기준으로 텍스트 길이 변경                                                                                |                                                                        |
| 자동 저장                | 스냅샷 저장 방식                                                                                          |                                                                        |
| 포커싱 변경               | 기존 페이지, 라인 단위 포커싱을 아래와 같이 변경<br>페이지X, 라인 뭉치 포커싱, 라인별 포커싱                                           | 라인 뭉치 = 라인의 다국어 포함 단위<br>한 - 각 라인 <br>중<br>일<br>---<br>전체를 라인 뭉치라고 표현함 |
| 한 페이지 내에 여러 언어 대본 보기 | 언어 선택<br>선택된 언어들에 대해 하나의 화면에서 대본을 확인할 수 있어야 함                                                      | 기존 언어를 하나의 페이지에서 볼 수 없어 병합 & 분리 여부를 판단하기 어려움                           |
| UI 최적화               | 페이지 행(페이지 표시, 수정 버튼 등) 제거<br>표시되는 대사 수를 늘려야 함<br>                                                  |                                                                        |
| 드래그 앤 드랍 키 이벤트       | 키 인벤트로 대사를 옮길 수 있어야 함                                                                              |                                                                        |
| 자막 수정 절차 간소화         | Confluence, Excel 처럼 수정이 뎁스 없이 바로 할 수 있어야 함                                                        |                                                                        |
| 인터미션 기능 추가           | PDF 정보 활용 자동 제공<br>대사 / 노래 / 인터미션으로 대사 타입 관리<br>대사와 같이 인터미션 안내 문구 추가할 수 있어야 함<br>사용자 시간 설정이 필요<br> | 인터미션 정의 : 막이 종료되고 10-20분 휴식을 갖음<br>사용자 시간 설정이 필요 없을 것으로 예상됨            |

### 공연시작

| 기능           | 설명                                                                                     | 비고                                            |
| ------------ | -------------------------------------------------------------------------------------- | --------------------------------------------- |
| 오디오 입력 게이지   | 오디오 입력 게이지를 통해 오디오 연결 상태 확인할 수 있게 함                                                    | 입력 마이크 시스템명 함께 표기                             |
| 부팅 로딩        | 의사결정 사항<br>1. 미리 준비하여 부팅 로빙바 기능 삭제(비용 발생)<br>2. 공연에 입장했을 때, 부팅함(비용 최소화 & 부팅 시간 필요)<br> | **정기하 의견**<br>공연 입장을 트리거로 부팅하는 것이 나쁘지 않다고 판단됨 |
| 오디오 입력 제어 토글 | 오디오 입력을 제어함                                                                            | 수동 모드와 사실상 동일<br>@김병준                         |
| 대본 선택        | 공연에 사용할 대본을 선택함                                                                        |                                               |
